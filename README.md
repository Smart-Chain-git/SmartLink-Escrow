## SmartLink demo with Vuetify & Typescript
[WIP]

### Usage

__Clone this repo__
```
git clone -b develop https://github.com/Smart-Chain-git/SmartLink-Escrow.git
```

__Install deps__
```
npm install
```

__Compiles and hot-reloads for development__
```
npm run serve
```
